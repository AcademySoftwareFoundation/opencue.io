outline.modules package¶

Submodules¶

outline.modules.shell module¶

Modules for executing arbitrary shell commands.

class outline.modules.shell.Shell(name, **args)¶

Provides a method of executing a shell command over an arbitrary frame range.

class outline.modules.shell.ShellSequence(name, **args)¶

A module for executing an array of shell commands.

class outline.modules.shell.ShellCommand(name, **args)¶

Provides a method of executing a single shell command. All instances of this class will result in a layer with a single frame regaurdless of what frame range the outline is launched to the cue with.

class outline.modules.shell.ShellScript(name, **args)¶

Copies the given script into frame’s session folder and executes it as a frame.

outline.modules.shell.shell(name, command, **args)¶

A factory method for building instances of Shell.

class outline.modules.shell.PyEval(name, code, **args)¶

Arbitrary python code execution.
